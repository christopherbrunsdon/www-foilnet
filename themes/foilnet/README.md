# FoilNet Theme

A professional Hugo theme for FoilNet Hair Salon featuring a dark aesthetic with gold accents.

## Features

- Black background with Cormorant Garamond display font and Inter body font
- Gold accent colour scheme for a premium salon feel
- Responsive layout for mobile and desktop
- Client-side search powered by Fuse.js
- Clean navigation with active-page highlighting
- Pricing tables, contact and banking detail page styles

## Configuration

See the main `hugo.toml` at the site root for all available parameters.
